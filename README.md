# 星火集 - 游戏创作者与创意行业综合平台

让创意闪耀光芒 ✨

## 关于项目

星火集是一个专为游戏创作者和创意工作者打造的综合平台，旨在连接创作者，激发灵感，共创未来。

### 特色功能

- 🎨 **作品展示** - 支持图片、视频、音频等多种格式
- 👥 **团队协作** - 与其他创作者组建团队，共同完成大型项目
- 💰 **收益变现** - 通过打赏、付费内容等多种方式获得回报

### 技术栈

- 前端：HTML5 + TailwindCSS
- 设计：响应式设计，自动适配PC/移动端
- 部署：Vercel
